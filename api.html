Dustbowl API¶

class dustbowl.api.Component¶

Base class for components.

Every component can declare what extension points it provides, as well as what extension points of other components it extends.

class dustbowl.api.ExtensionPoint(interface)¶

Marker class for extension points in components.

class dustbowl.api.implements¶

Can be used in the class definiton of Component subclasses to declare the extension points that are extended.

class dustbowl.api.Interface¶

Marker base class for extension point interfaces.

class dustbowl.api.IShellCommandProvider¶
match(cmd)¶

Returns True if the provider handles the provided command. False otherwise.

run(cmd, *args, **kwargs)¶

Invokes the given command

class dustbowl.api.IShellConsoleObjectProvider¶
get_console_objects()¶

Return an iterable of 2-tuples of (key, value) pairs.

The value will be injected in the console context as the given key
